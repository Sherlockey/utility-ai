Gameplay Controls
- Open Utility Display - Escape
- Pause - `
- 1X Speed - 1
- 2X Speed - 2
- 3X Speed - 3
- Move Camera Up - Up Arrow
- Move Camera Down - Down Arrow
- Move Camera Left - Left Arrow
- Move Camera Down - Down Arrow
- Center Camera On Active Combatant - Enter

Debug Controls
- Show Influence Map and Movement Utility - F1
- Show Decision Log - F2
- Set Team Control To None - Z
- Set Team Control To Ally - X
- Set Team Control To Enemy - C
- Set Team Control To All - V
- Move Combatant Up - W
- Move Combatant Down - S
- Move Combatant Left - A
- Move Combatant Right - D
- Use Combatant Ability 1 - Q
- Use Combatant Ability 2 - E
- End Turn - Space
